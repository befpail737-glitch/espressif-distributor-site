ESP32-C3

Cost-Effective RISC-V MCU with Wi-Fi & Bluetooth 5 LE

Home / Products / ESP32-C3

Product Overview

The ESP32-C3 is a safe, secure, and stable MCU with Wi-Fi and Bluetooth 5 (LE) connectivity. Built around a RISC-V single-core processor, it provides an optimal balance of power consumption, security, and cost-effectiveness for a wide range of IoT applications.

32-bit RISC-V Single Core up to 160 MHz
Wi-Fi 4 + Bluetooth 5 LE
Advanced Security Features
Cost-Effective Solution
